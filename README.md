
<h1>Dice-Rolling-Game</h1>

<h2>Description</h2>
This is a simple Python command-line dice rolling simulator. It allows users to roll one or more six-sided dice as many times as they want, and keeps track of how many total dice rolls they've made during the session.
<br />

<h2>Features</h2>

- <b>Roll as many dice as you like in one go</b> 
- <b>Keeps a running count of how many individual dice have been rolled</b>
- <b>Handles invalid inputs gracefully (e.g. non-numeric or negative values)</b>
- <b>Simple and interactive CLI experience</b>

<h2>Languages and libraries Used</h2>

- <b>Python</b> 
- <b>random module</b>
